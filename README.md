# CryptoMarketAPI
CryptoMarket With Node Js And Mongo DB
## Installation
- Download Node.js 10.xxx or higher
- Create Mongo DB Account
- Download Postman
### How to use it
- npm install
- npm start
#### Environment
- export API_CRYPTO=
- export API_KEY_CRYPTO=
- export JWT_KEY=
- export DB_URI=
- export DB_NAME=
- export PORT=
- export IS_AES=
- export CIPHER_KEY=
- export IV_KEY=
- export CLIENT_KEY=
